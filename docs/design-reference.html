Hoppin' — Design Reference Specifications

This file contains the exact HTML/CSS mockups approved during the product design phase. Use these as the authoritative visual reference when implementing React + Tailwind components. Every hex value, pixel measurement, and structural pattern in this file is intentional and approved.

Note: The subway station card component has a separate dedicated design prompt that takes priority over the general patterns in this file for that specific component.
